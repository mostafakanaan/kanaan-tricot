# Kanaan Tricot — GitHub Pages DNS Test

A super‑small static site to verify your custom domain on GitHub Pages.

## Local preview

Just open `index.html` in your browser.

## Deploy

1. Create a repo on GitHub (e.g. `kanaan-tricot`).
2. Push these files to the `main` branch.
3. In **Settings → Pages**, choose **Deploy from a branch** → `main` / `/root`.
4. (Optional) Add your **Custom domain** and enable **Enforce HTTPS**.

For DNS records and custom domain docs, see GitHub’s official guide.
